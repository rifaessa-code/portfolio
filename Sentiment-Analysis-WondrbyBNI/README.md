# Sentiment Analysis Wondrb by BNI (mobile banking app) with Modified K-Nearest Neighbor

Tentang proyek ini:
Proyek ini merupakan skripsi saya pada kuliah S1 untuk menyelesaikan sarjana statistika.

Sedikit tentang MK-NN:
MK-NN merupakan kembangan dari algoritma KNN yang berfungsi untuk meningkatkan akurasi pada klasifikasi.

Kenapa menggunakan Wondr by BNI sebagai topik?
Alasan pemilihan wondr by BNI sebagai topiknya karena pada saat itu wondr by BNI baru saja launching dan mendapatkan penuruan rating yang signifikan selama periode penelitian.

Bagaimana akurasi pada hasil klasifikasi?
Akurasi yang dihasilkan sebesar 87,24%.

Sumber data penelitian?
Data bersumber dari Google Play Store dan menggunakan scrapping untuk mengambil data rating serta komentar dari para user.
